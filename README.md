# 🎓 Mentor-Mentee Platform (MentorHub)

A full-stack web application where mentors assign weekly challenges, mentees submit solutions, mentors provide feedback, and the system tracks progress across multiple sessions.

---

## 🚀 Features

- 👨‍🏫 Mentor Dashboard
  - Create sessions (Week-wise)
  - Create challenges (title, description, subject, difficulty, deadline)
  - View submissions
  - Provide feedback and ⭐ ratings

- 👨‍🎓 Mentee Dashboard
  - View challenges
  - Submit solutions
  - View feedback and ratings
  - Track progress history

- 📊 Data Visualization
  - Progress charts (Reviewed vs Pending)
  - Submission trends
  - Performance analysis

- 🏆 Leaderboard
  - Rank mentees based on activity and performance
